# Vision Education Chatbot (Streamlit)

A simple educational chatbot that helps users describe blurry near vs distance vision, asks two follow-up questions (one/both eyes, sudden/gradual), and provides safe next-step guidance. Includes safety guardrails for warning signs like flashes, floaters, and eye pain.

## Safety Note
This app provides educational guidance only and is not a medical diagnosis. If you have sudden vision changes, eye pain, flashes, or new floaters, seek urgent eye care.
